# dSound

## I. Set Up
### 1. Clone project
```
  git clone git@github.com:DAMHONGDUC/dSound.git
```
### 2. Install packages
- Open terminal in project folder and run:
```
  yarn install
```

### 3. Run app
- Open terminal in project folder and run these command:
```
  yarn start
```
```
  yarn android
```

## II. Tech stack
- React Native
- Firebase
- React Native Track Player
- Zing Mp3 API

## III. Release
- Android [apk](release/dsound.apk)
- iOS (not available)
